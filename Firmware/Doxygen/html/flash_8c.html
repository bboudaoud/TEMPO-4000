Write a single sector to the flash card.

Attempt to write a full sector (including computed checksum)

Provides a clean read wrapper separating the rest of the codebase from the MMC library.

Parameters
sectorNumThe sector number to read in (0 to 4194303 for a 2GB MMC card with 512B sector sizes).
[in,out]sectorPointer to sector to write. Will have a newly computed checksum written into it. Most likely you will be casting a more specific sector type or a simple array to this interface
Return values
0success
-1failure (MMC library returned an error)
See Also
secureFlashWrite() for more thorough write validation
